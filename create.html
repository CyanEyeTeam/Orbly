Recommended Settings

  • Use square images (46×46 pixels recommended) for each state

  • The download will automatically format as 55×162 pixels
  • Make sure all three states have the same dimensions

Installation Guide

  1. Download your custom Start Orb using the button above.
  2. Install StartAllBack on your Windows device.
  3. Open the program settings and navigate to the Taskbar customization section.
  4. Under 'Visual Style' select "+" and browse to your downloaded orb file.
  5. Apply the changes and enjoy your custom Start button!

Note: To use custom Start Orbs, you need third-party software as Windows doesn't natively support changing the Start button.
